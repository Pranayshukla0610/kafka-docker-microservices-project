# Kafka Microservices Project 🚀

## 📌 Overview
This project demonstrates a real-time data streaming pipeline using Apache Kafka with a producer-consumer architecture.

## ⚙️ Tech Stack
- Python
- Apache Kafka
- Docker
- FastAPI (optional)
- Kafka-Python

## 🏗️ Architecture
Producer → Kafka Topic → Consumer

## 🚀 Features
- Real-time message streaming
- Kafka-based microservices communication
- Fraud detection logic
- Dockerized Kafka setup
